Sudoku Rules

  • Sudoku Rules
    • Game is played on a 9x9 grid composed of 9 3x3 sub-grids
    • Game table is pre-populated with integers between 1 and 9
      leaving open spots to be filled by the player
    • To Win
      • Each 3x3 grid will be filled with 1-9 a single time
      • Each row and column in the table will be filled with
        1-9 a single time per row and column

Preliminary Software Design

  • Objects List
    • Space
      • Value being held in the Space Currently
      • Fixed variable for all pre-populated Spaces
      • Methods getSpace and setSpace
      • Method valid to determine the validity of incoming space values
      • Methods to let the associated Group object know when a value has changed
    • Group
      • Representation of the 3x3 group containing references to Space objects
      • Function to determine whether the Group object references all valid input
        and return boolean to aid solving the Sudoku Table
    • Table
      • Generation method to pre-populate the table with some values
      • Clear method to clear the table of user supplied input
      • Column and Row value trackers represented as arrays
      • Method to determine the validity of the whole table by
        determining the validity of the contained groups, columns, and rows
